# Contribuindo

Este repositório é um protótipo de produto. As contribuições devem preservar clareza visual, privacidade e coerência com o PRD.

## Fluxo recomendado

1. Crie uma branch curta e descritiva.
2. Faça alterações pequenas e verificáveis.
3. Rode `npm run build` antes de abrir PR.
4. Para mudanças visuais, inclua screenshot ou GIF.
5. Explique o impacto para usuário, não apenas o arquivo alterado.

## Padrões

- Interface em português do Brasil.
- Não prometer integração real quando ela ainda é demonstrativa.
- Não adicionar scraping de LinkedIn ou redes sociais.
- Manter dados de demo separados de fluxos reais.
- Preferir componentes e estilos já existentes.
- Preservar acessibilidade básica: contraste, foco e textos claros.

## Checklist antes de enviar

- [ ] Build passou.
- [ ] A tela alterada foi testada no navegador.
- [ ] Não há erros relevantes no console.
- [ ] O README/docs foram atualizados se a mudança afetar apresentação ou arquitetura.
- [ ] Não foram adicionados segredos, tokens ou dados reais sensíveis.
